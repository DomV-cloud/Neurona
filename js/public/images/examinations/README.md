# Medical Examination Images

## Instructions:

1. Save the brain MRI image as: `brain-mri-sample.jpg`
2. The image will automatically appear in the patient details page
3. Click on the image to view it in full-screen mode with zoom and rotation controls

## Supported formats:

- JPG/JPEG
- PNG
- WebP

## Recommended specifications:

- Minimum resolution: 512x512 pixels
- Maximum file size: 5MB
- Square aspect ratio preferred for optimal display
